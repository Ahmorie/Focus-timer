# Focus Timer

Application minimaliste de timer de focus avec modes Pomodoro et countdown personnalisé.

## Fonctionnalités

- 🕐 Horloge en temps réel
- 🍅 Mode Pomodoro configurable
- ⏱️ Timer countdown personnalisable
- 🔊 Effets sonores à la fin des sessions
- 🎨 Design minimaliste noir et blanc

## Développement

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build de production
npm run build
```

## Déploiement

L'application est prête pour être déployée sur Vercel :

```bash
# Déployer avec Vercel CLI
vercel
```

Ou connectez simplement votre dépôt GitHub à Vercel pour un déploiement automatique.

## Technologies

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Web Audio API
